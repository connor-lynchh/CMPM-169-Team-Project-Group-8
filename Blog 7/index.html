Review: Viritual

Today’s review will be about a niche mobile game I was recommended by a friend. It’s called Viritual, and it’s honestly pretty badass. It’s an AR horror game where you have to track down rifts in reality, which demons can speak through to give you instructions for how to perform a ritual. I almost think it uses some kind of hypnosis or neuro-linguistic programming or something, because it’s super immersive— like, with some of the demon voices I get this chill down the back of my neck while they’re talking, and it really feels like there’s something standing right behind me. Last night I totally zoned out while playing and ended up walking twice as far as I meant to without even realizing. Whatever trick they’re using with the audio, they’re doing a great job.

My only critique about the game would be that it’s got some sort of setting that makes it run in the background all the time, and I can’t figure out how to turn it off. They could probably use some improvements to the UI in general, actually, it’s a bit hard to navigate. Anyway, every so often a clue will pop up while I have the app closed and I’m doing other things. It’s fun sometimes, but I’d like it if I could turn that feature off, especially when I’m at work. Other than that, though, I really don’t have any complaints.

Apparently, there’s a big event coming up in the game, so if you haven’t downloaded the app yet, you should do so now! I heard there’s been an issue where some people have trouble finding it in the app store— it’s not always showing up in search results for some reason. Anyway, if you’re trying to download it and can’t find it in the app store, DM me and I’ll send you a referral link. I don’t want to post it here because I’ve gotten some rude feedback about including referral links in my reviews.

Anyway, hope y’all are having a great day, and I look forward to seeing you in-game!
